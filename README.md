# Análisis de delitos en Guanajuato 2020-2025 (SESNSP)

Este repositorio contiene un ejercicio de análisis exploratorio de datos públicos del Secretariado Ejecutivo del Sistema Nacional de Seguridad Pública (SESNSP), enfocado en la incidencia delictiva municipal del estado de Guanajuato.

## Fuente de datos
- Incidencia delictiva municipal (SESNSP)
- Periodo: 2015–2025

## Objetivo
Explorar tendencias temporales y patrones territoriales de la incidencia delictiva como ejercicio de análisis de datos y visualización.

## Estado del proyecto
Proyecto borrador en desarrollo y revisión de clientes.
